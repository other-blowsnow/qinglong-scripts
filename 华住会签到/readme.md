签到得积分，可以兑换华住酒店优惠券

## 使用方法
- 打开 “华住会” 小程序，抓包，找 cookie里面有 userToken=xxxxxxxxxx的值
- 设置环境变量，HZH_TOKEN='上面获取的userToken'，多个账号使用 @分割
